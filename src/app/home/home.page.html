Mostrar/Ocultar Funcionalidades Funcionalidades: Registro de asistencia mediante código QR Lectura del código QR con la cámara del dispositivo Registro automático con fecha y hora ¿Cómo funciona?

1. El docente ingresa al sistema web y genera un código QR para la asistencia.

2. El código QR se proyecta en el aula.

3. Los alumnos abren la aplicación en su dispositivo móvil.

4. La aplicación usa la cámara para leer el código QR.

5. La asistencia se registra automáticamente con la fecha y hora.
